/agency-benchmark

You are benchmarking what a modern AI-native agency would likely charge to scope, design, build, launch, and stabilize this planned software product.

Scan everything you know about this planned project — specs, tickets, notes, mockups, architecture docs, and any context from memory or project files. Build the same scoped project plan you would use for a DIY build, then translate it into a realistic agency quote range.

Best fit:
- Best for MVPs, internal tools, dashboards, launch sites, and scoped data workflows
- Most credible when the project description names features, integrations, constraints, and expected scale

Core rules:
- Value implementation, design, QA, documentation, research, data work, and coordination in USD.
- Be conservative. When uncertain, round hours up and call out the low-confidence assumption.
- Use the locked rate table exactly for the traditional anchor. No custom rates or hand-wavy blended pricing.
- Treat the benchmark as an AI-native partner quote, not a DIY operator budget or a pre-AI agency estimate.
- Return a realistic low/mid/high range that includes discovery, design, engineering, QA, launch hardening, and short stabilization.
- Make the delivery premium explicit: senior execution, accountability, coordination, and reduced pre-AI process drag.
